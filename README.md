## Installation

1. Extract all files to a folder
2. Open `index.html` in a modern web browser (Chrome, Edge, Firefox)
3. No server installation required - runs entirely in the browser

## Usage

### Navigation
- Use the left sidebar to navigate between different sections
- Click on any menu item to view the corresponding data

### Filtering & Search
- Use the search boxes to filter users, groups, or licenses
- Select dropdown filters to view specific categories
- Global search available in the header

### Data Export
- Click "Export" buttons to download data (simulated in prototype)
- Generate reports from the Reports section

### Interactive Features
- View user details by clicking "View" buttons
- Hover over cards for visual feedback
- Charts update based on data changes
